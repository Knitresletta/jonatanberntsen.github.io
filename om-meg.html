Bakgrunn

Gjennom mange år har jeg drevet frivillig kursvirksomhet for barn og ungdom — innen musikk-samspill, foto og video, programmering og klatring. Å jobbe med unge mennesker i praktiske, skapende sammenhenger er noe jeg brenner for, og det har gitt meg solid erfaring med å formidle komplekse ting på en måte som faktisk sitter.

Jeg sitter i styret i ORKK klatreklubb med ansvar for medlemmer, arrangementer og barneklatring.

For noen år siden begynte jeg å kode. Det startet med små skript for å løse irriterende hverdagsproblemer, og endte med Flask-apper, TUI-dashbord, AI-botter og en iPhone-app. Nå sikter jeg mot en jobb som backendutvikler.

Hva jeg jobber med

Jeg tar Boot.dev-kurset i backend-utvikling og jobber parallelt med egne prosjekter som faktisk brukes — en MTG-lagersjekker, en Instagram-bot for klatreklubb, og verktøy for barn og unge i kurssetting. Jeg foretrekker kode som er lesbar og vedlikeholdbar over kode som er smart.

Teknologier

Python Flask HTML / CSS JavaScript Git Textual (TUI) BeautifulSoup4 REST API Claude API SwiftUI Lua Linux / Bazzite

Utenfor koden

Klatring, musikk, Ableton, D&D, Magic: The Gathering og for mye tid brukt på å optimalisere ting som allerede fungerer godt nok. Jeg tror på varme fellesskap, direkte kommunikasjon og at humor er undervurdert i teknologibransjen.
