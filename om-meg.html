// om meg

Jonatan Berntsen

Lærer og ungdomsarbeider som fant veien inn i kode — og ikke har tenkt å gå tilbake.

Bakgrunn

Jeg jobber som lærer og ungdomsarbeider i Norge, og har i mange år jobbet i skjæringspunktet mellom mennesker, systemer og praktisk problemløsning. Jeg leder ORKK klatreklubb og bruker mye tid på å hjelpe unge mennesker med å vokse — både klatreteknisk og som mennesker.

For noen år siden begynte jeg å kode. Det startet med små skript for å løse irriterende hverdagsproblemer, og endte med Flask-apper, TUI-dashbord, AI-botter og en iPhone-app. Nå sikter jeg mot en jobb som backendutvikler.

Hva jeg jobber med

Jeg tar Boot.dev-kurset i backend-utvikling og jobber parallelt med egne prosjekter som faktisk brukes — en MTG-lagersjekker, en Instagram-bot for klatreklubb, og verktøy for klasserommet. Jeg foretrekker kode som er lesbar og vedlikeholdbar over kode som er smart.

Teknologier

Python Flask HTML / CSS JavaScript Git Textual (TUI) BeautifulSoup4 REST API Claude API SwiftUI Lua Linux / Bazzite

Utenfor koden

Klatring, musikk, D&D, Magic: The Gathering og for mye tid brukt på å optimalisere ting som allerede fungerer godt nok. Jeg tror på varme fellesskap, direkte kommunikasjon og at humor er undervurdert i teknologibransjen.
